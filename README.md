# Voety

Voety - Simple Football Lineup Builder

## How to run?

If you are a total beginner to this, start here!

1. Install

   ```
   git clone https://github.com/alperunal/voety.git
   cd voety
   yarn install
   ```

2. Run

   `yarn start`

3. Build

   `yarn build`

4. Test

   `yarn test`
